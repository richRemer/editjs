EditJS is a JavaScript code editor based on CodeMirror.

Installing
==========

Clone Repository
----------------
```sh
git clone git@github.com:richRemer/editjs.git
```

Navigate to Repository
----------------------
```sh
cd editjs
```

Build Using make
----------------
```sh
make
```

Install Using make
------------------
```
sudo make install
```

Uninstalling
============
Navigate to Repository
----------------------
```sh
cd editjs
```

Uninstall Using make
--------------------
```sh
sudo make uninstall
```
